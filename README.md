## Installation instructions
 ### Clone the repo
  ```bash
  git clone https://github.com/A-Unix/YoutubeViewer.git
  ```
 ### Move to cloned folder
  ```bash
  cd YoutubeViewer
  ```
 ### Get permissions
  ```bash
  chmod +x *
  ```
 ### Install dependencies
  ```bash
  ./setup.sh
  ```
 ### Run the main file
  ```bash
  python3 main.py
  ```
Enjoy :)
